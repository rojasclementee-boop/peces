# 🐠 Especies de Peces - Página Web Educativa

Una página web interactiva que explora 10 especies fascinantes de peces del océano.

## 📋 Contenido

La página incluye información detallada sobre:

1. **Pez Payaso** - Simbiontes de anémonas
2. **Tiburón Blanco** - Depredador cúspide
3. **Pez Cirujano Azul** - Herviboro del arrecife
4. **Atún Rojo del Atlántico** - Migradores oceánicos
5. **Mero Gigante** - Gigante del arrecife
6. **Pez Globo** - Con defensa química
7. **Pez Ángel Emperador** - Hermafrodita secuencial
8. **Salmón Atlántico** - Migradores anádromos
9. **Pez Martillo** - Con electrorrecepción
10. **Pez Vela** - El pez más rápido

## 📊 Para cada especie se incluye:

- ✅ Características físicas y hábitat
- ✅ Información de reproducción y ciclo de vida
- ✅ Aporte ecológico al ecosistema marino

## 🎨 Características del Sitio

- Diseño moderno y responsivo
- Navegación suave
- Colores oceánicos (azul, celeste, blanco)
- Animaciones interactivas
- Formulario de contacto
- Compatible con dispositivos móviles
